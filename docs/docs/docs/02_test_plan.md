1. Stock controller and supply – 100% brightness
2. Stock controller and supply – 50% brightness
3. Custom controller and supply – 50% brightness
4. Custom controller and supply – brightness cap (~95%)


## Audio Tests

1. Stock system – 100% brightness
2. Stock system – 50% brightness
3. Custom controller – 50% brightness
